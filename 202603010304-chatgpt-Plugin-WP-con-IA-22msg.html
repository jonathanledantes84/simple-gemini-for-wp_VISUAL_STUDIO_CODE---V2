== Description ==
Simple Gemini for WP permite importar contenido automáticamente desde RSS, X y Instagram y generar posts usando los modelos Gemini. Incluye opciones para configurar prompt personalizado, límite diario, análisis de imágenes (Gemini Vision), y distintas formas de notificación.

== Installation ==
1. Subir la carpeta `simple-gemini-for-wp` a `/wp-content/plugins/`.
2. Activar el plugin desde el panel de Plugins.
3. Ir a Ajustes → Simple Gemini para configurar: API keys, prompt, categorías, límite diario y notificaciones.
4. Probar con "Generar post ahora".

== Frequently Asked Questions ==

= ¿Necesito pagar por usar Gemini? =

Google ofrece un tier gratuito generoso en Google AI Studio. Para un portal con 1–5 posts diarios normalmente no hay costo.

= ¿Dónde consigo la API Key? =

1. Ir a Google AI Studio  
2. Crear o iniciar sesión con tu cuenta Google  
3. Generar API Key  
4. Pegala en Ajustes → Simple Gemini  

= ¿Funciona en hosting compartido? =

Sí. El plugin usa funciones nativas de WordPress como `wp_remote_post()` y `fetch_feed()`. No requiere Composer ni dependencias externas.

= ¿Puedo usarlo sin el cron automático? =

Sí. Podés usar el botón manual en el editor Gutenberg sin activar el cron.

= ¿Qué pasa si falla Gemini? =

El plugin realiza hasta 2 intentos automáticos. Si falla, queda registrado en logs y se envía notificación por email.
