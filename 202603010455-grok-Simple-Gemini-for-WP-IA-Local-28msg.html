Plugins WordPress que integran Grok (xAI) de forma nativa

  1. MxChat – AI Chatbot & Content Generation (gratuito con opciones Pro)
    • El más completo y popular para multi-modelos.
    • Soporta directamente xAI Grok (incluyendo Grok-4, Grok-3, Grok-3 Fast, Grok-4.1 Fast Reasoning, etc.).
    • Funciones: Chatbot en frontend, generación de contenido (posts, resúmenes), entrenamiento con tu contenido web (RAG), integración WooCommerce si vendés algo.

    • Link: Disponible en WordPress.org → Buscá "MxChat".
    • Ventaja para vos: Cambiás entre Gemini y Grok en el mismo plugin (ideal si querés probar Grok sin abandonar Gemini).
  2. AI Power (anteriormente GPT3 AI Content Generator)
    • Uno de los plugins AI más descargados.
    • Soporta xAI Grok para texto e imágenes (Grok genera imágenes muy buenas y "sin restricciones").
    • Automatización: Genera posts masivos desde keywords/CSV, cron jobs, internal linking automático.
    • Usuarios destacan que Grok da resultados "muy superiores" en calidad y originalidad vs. otros modelos.
    • Perfecto si querés migrar tu flujo de auto-posts de Gemini a Grok (o híbrido).
    • Link: wordpress.org/plugins/gpt3-ai-content-generator
  3. WP Sheet Editor – AI (extensión paga, pero potente)
    • Genera posts completos en masa desde spreadsheet.
    • Conecta directo a Grok-4 vía OpenRouter (o xAI API).
    • Ideal para portales de noticias: Generás 5–10 notas diarias sobre Alto Valle/Patagonia con prompts personalizados.
    • Tutoriales recientes muestran exactamente cómo usarlo con Grok para posts WP.
  4. Otros con soporte Grok (chatbots o summarizers)
    • Baachal AI Chatbot → WooCommerce + Grok para búsqueda conversacional de productos/noticias.
    • Sumtics – AI Summarizer → Genera TL;DR de posts con Grok (útil para resúmenes rápidos de noticias locales).
    • Limb AI Chatbot o WPBot → Multi-modelo (incluye Grok vía OpenRouter), con RAG para entrenar con tu sitio.

  1. Ingresá al xAI Console Abrí tu navegador y andá directo a: https://console.x.ai (o https://console.x.ai/team/default/api-keys para ir directo a keys una vez logueado).
  2. Creá o iniciá sesión
    • Usá tu cuenta de X para loguearte (es el método principal).
    • Si es tu primera vez: Completá el onboarding rápido (info de developer, use case como "WordPress content generation for local news", aceptá términos). Puede tardar 24–48 hs en activarse si hay revisión, pero suele ser inmediato.
  3. Agregá créditos (necesario para usar el API)
    • Una vez dentro del console, andá a Billing o Add Credits.
    • Cargá un monto mínimo (ej. $10–25). Nuevos suelen tener $25 free credits automáticos.
    • Precios aproximados 2026 (por millón de tokens):
      • Grok-4.1 Fast (recomendado para tu uso local, rápido y barato): ~$0.20 input / $0.50 output.

      • Contexto gigante: hasta 2M tokens en algunos modelos → ideal para resumir RSS largos de Río Negro o Patagonia.
  4. Generá la API Key
    • En el dashboard, andá a API Keys (o directamente https://console.x.ai/team/default/api-keys).
    • Hacé clic en Create API Key o Generate new key.
    • Dale un nombre descriptivo (ej: "WP Noticias Roca - Grok Auto Posts").

    • Si adaptás tu Simple Gemini for WP: Cambiá las llamadas de Gemini (endpoint https://generativelanguage.googleapis.com) por xAI:
      • Base URL: https://api.x.ai/v1
      • Model: grok-4-1-fast o similar
      • Auth: Bearer + tu API key
      • Es casi plug-and-play con SDKs OpenAI-compatible (usa openai Python lib cambiando base_url).
    • Probá con un prompt simple: "Generá una noticia corta sobre clima en General Roca hoy".
