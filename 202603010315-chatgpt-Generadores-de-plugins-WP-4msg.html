  1. Generadores de esqueleto (crean la estructura .zip del plugin en un clic)

  • WPPB — genera un boilerplate personalizado y te da el zip para descargar. [WordPress Plugin Boilerplate]

  • WordPress Plugin Generator (wordpressplugingenerator.com) — freemium, rápido para plantillas básicas. [Generador de Plugins de WordPress]

  • WPSandbox — WP Plugin Generator — subís un snippet y te empaqueta el plugin listo para instalar. [WPSandbox]

  1. Plataformas tipo “Replit” para editar / probar en la nube (IDE + entorno)

  • Replit o GitHub Codespaces + @wordpress/env te permiten editar y levantar un WordPress de desarrollo en la nube para probar el plugin en vivo. Hay tutoriales que muestran exactamente ese flujo. [YouTube]

  1. Herramientas low-code / AI que ayudan a generar código o proyectos WordPress (más orientadas a sitios completos, pero útiles para idear funcionalidades)

  • Telex (Automattic) — entorno asistido por IA para proyectos WordPress. [Telex]

  • 10Web y WordPress.com ofrecen asistentes/AI para generar sitios y partes de código; no son “generador de plugin” puro, pero se pueden usar para prototipar ideas y luego convertir a plugin. [TechRadar]

Recomendación práctica (flujo rápido que suele funcionar bien):

  1. Generás el esqueleto con WPPB o wordpressplugingenerator para tener la estructura básica.

  2. Abrís ese código en Replit o Codespaces, instalás @wordpress/env o un contenedor Docker para levantar un WP de desarrollo y probás las funciones.

  3. Si querés ayuda para escribir la lógica, podés prototiparla con herramientas IA (Telex / 10Web) y luego integrarla al plugin.

// Meta Cloud WhatsApp send (simple)
function sgfw_send_whatsapp( $phone_id, $token, $message ) {
$endpoint = "https://graph.facebook.com/v17.0/{$phone_id}/messages";
$body = array(
'messaging_product' => 'whatsapp',
'to' => (string) $phone_id, // user should replace with target number in settings in future versions
'type' => 'text',
'text' => array( 'body' => $message )
);
$args = array(
'headers' => array( 'Authorization' => 'Bearer ' . $token, 'Content-Type' => 'application/json' ),
'body' => wp_json_encode( $body ),
'timeout' => 20,
);
$res = wp_remote_post( $endpoint, $args );
if ( is_wp_error( $res ) ) return false;
$code = wp_remote_retrieve_response_code( $res );
return ( $code >= 200 && $code < 300 );
}
// Telegram send via Bot API (supports photo if thumbnail id provided)
function sgfw_send_telegram( $bot_token, $chat_id, $message, $thumb_attachment_id = 0 ) {
$endpoint = "https://api.telegram.org/bot{$bot_token}/sendMessage";
$args = array(
'body' => array(
'chat_id' => $chat_id,
'text' => $message,
'disable_web_page_preview' => false,
),
'timeout' => 20,
);
$res = wp_remote_post( $endpoint, $args );
if ( is_wp_error( $res ) ) return false;
$code = wp_remote_retrieve_response_code( $res );
return ( $code >= 200 && $code < 300 );
}

'methods' => 'POST',

'permission_callback' => function() { return current_user_can( 'edit_posts' ); }

function sgfw_rest_generate( $request ) {
$params = $request->get_json_params();
$prompt = sanitize_textarea_field( $params['prompt'] ?? '' );
if ( empty( $prompt ) ) return new WP_Error( 'no_prompt', 'No se recibió prompt' );
$generated = sgfw_call_gemini( $prompt );
if ( is_wp_error( $generated ) ) return $generated;
return rest_ensure_response( $generated );
}
